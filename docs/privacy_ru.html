Политика конфиденциальности

Приложение «Seasons» для РУДН · Последнее обновление: 30 января 2026 г.

1. Введение

Приложение «Seasons» разработано для студентов и сотрудников Российского университета дружбы народов (РУДН) и предназначено для участия в университетских голосованиях и конкурсах.

Мы уважаем вашу конфиденциальность и обязуемся защищать ваши персональные данные в соответствии с законодательством Российской Федерации.

2. Какие данные мы собираем

Мы НЕ собираем и НЕ храним:
  • Ваши пароли (аутентификация через официальный сайт РУДН)
  • Данные геолокации
  • Контакты или фотографии с вашего устройства
  • Данные для рекламных целей

Приложение использует:

  • Сессионный токен — для авторизации на сервере seasons.rudn.ru
  • Настройки языка — для персонализации интерфейса (русский/английский)
  • Идентификатор устройства — для push-уведомлений о голосованиях
  • Опциональная диагностическая телеметрия сбоев — технический тип события, версия приложения, платформа и экран; отключено по умолчанию и отправляется только при явном включении для расследования релизных проблем

3. Как мы используем данные

Собранные данные используются исключительно для:

  • Авторизации в системе голосований РУДН
  • Отображения доступных голосований и их результатов
  • Отправки уведомлений о начале новых голосований

4. Хранение данных

Сессионный токен хранится локально на вашем устройстве в защищённом хранилище (Keychain для iOS, EncryptedSharedPreferences для Android) и не передаётся третьим лицам.

При выходе из аккаунта все локальные данные удаляются.

5. Передача данных третьим лицам

Мы не передаём ваши данные третьим лицам, за исключением:

  • Серверов РУДН (seasons.rudn.ru) — для работы системы голосований
  • Первичного защищённого relay-узла РУДН — только для опциональной диагностической телеметрии сбоев при явном включении
  • Случаев, предусмотренных законодательством РФ

6. Ваши права

Вы имеете право:

  • Выйти из аккаунта и удалить все локальные данные
  • Отключить push-уведомления в настройках устройства
  • Удалить приложение, что автоматически очистит все данные

7. Изменения в политике

Мы можем обновлять данную политику. Актуальная версия всегда доступна по этой ссылке. При существенных изменениях мы уведомим вас через приложение.
